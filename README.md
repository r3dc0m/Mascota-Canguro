# Mascota Canguro
## Descripción
Este es un prototipo estético de una aplicación web dirigida a personas que requieren de un servicio de acogida de sus mascotas durante sus vacaciones o para personas que amen a los animales y deseen monetizar un espacio para ellos.

Idealmente estas personas tendrán que dedicar cierta fracción de tiempo a los cuidados básicos o especiales que requieran los propietarios. 

El proyecto podría extenderse a otros servicios que permitieran transportar animales, pasear mascotas o cuidar directamente de las mascotas dentro de los espacios de los propietarios.

Según el sistema de reservas, su cuantía y/o calidad, la cuota del propietario del espacio se verá reducida en un porcentaje establecido.
## Estructura del Prototipo
La vista principal describe de forma breve cual es el servicio que ofrece la página web y ésta compone la página de inicio.
* **index.html:** Descripción de los servicios.
* **search.html:** Interfaz principal de búsqueda.
* **services.html:** Otros servicios.
## Tecnologías Utilizadas
* HTML5
* CSS3
## Enfoque Técnico
El desarrollo inicial se enfoca en un diseño adaptado a móvil, a medida que el largo de ventana incrementa, los contenedores permiten ver más tarjetas y un margen lateral se forma para facilitar la lectura de forma más centrada.
Animaciones perceptibles con puntero, que no existen en su forma móvil.
* Diseño Mobile First.
* Uso de selectores básicos y combinados.
* Uso de diferentes valores de Display y Position.
